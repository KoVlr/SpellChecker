SpellChecker - проект исправления опечаток и ошибок в словах, набранных пользователем.

Исправление опечаток происходит в файле SpellChecker.py, в котором вызывается функция из файла SpellChecker.cpp.
Для работы этой программы нужны файлы substitution.csv и voctrie.csv, которые являются результатом обучения модели, происходящего при выполнении файла ModelTraining.py.
Входными данными для обучения модели является файл dataset.txt с большим количеством сгенерированного пользователями текста, а также файл vocwords.txt, в котором содержится набор словарных слов.
